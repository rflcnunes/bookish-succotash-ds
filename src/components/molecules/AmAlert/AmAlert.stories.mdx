# AmAlert

The component `AmAlert` is used to display a message to the user.

## Props
The component AmAlert has the following properties:

- `variation` (String, opcional): A string prop that defines the type of the message, with the possible values of "warning", "error", "success", "info", or "helper". The default value is "info". The prop also includes a validator function that checks if the provided value is one of the accepted values.
- `title` (String, opcional): A string prop that defines the title of the message. The default value is an empty string.
- `text` (String, opcional): A string prop that defines the main text of the message. The default value is an empty string.
- `link` (String, opcional): A string prop that defines a link that can be included in the message. The default value is an empty string.
- `linkLabel` (String, opcional): A string prop that defines the label for the link. The default value is "Learn more".

## Usage
```vue
<template>
  <div>
    <AmAlert variant="success" title="Success Title" text="This is a success message!" />
    <AmAlert variant="warning" title="Warning Title" text="This is a warning message!" />
    <AmAlert variant="error" title="Error Title" text="This is an error message!" />
  </div>
</template>

<script>
import AmAlert from '@/components/molecules/AmAlert/AmAlert.vue';

export default {
  components: {
    AmAlert,
  },
};
</script>
```
