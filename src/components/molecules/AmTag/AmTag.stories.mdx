# AmTag

The `AmTag` component is used to display a tag.

## Props
The component AmTag has the following props:

- `text` (String, required): The text to display in the tag.
- `variant` (String, optional): The variant of the tag. Possible values are `gray`, `darkGray`, `blue`, `darkBlue`, `yellow`, `orange`, `purple`, `darkPurple`, `red`, `darkRed`, `green`, and `darkGreen`. Default value is `gray`.
