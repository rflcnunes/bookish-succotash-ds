# AmButton
The component ``AmButton`` is a customizable button in Vue.js, which can be used to create buttons with different sizes, style variations, background colors and labels.


## Props
The component `AmButton` has the following properties:

- `size` (String, optional): Defines the size of the button. The possible values are small, medium, and large. The default value is medium.
- `variant` (String, optional): Defines the style of the button. The possible values are primary, secondary, dashed, purple, error, outlined, and green. The default value is primary.
- `label` (String, optional): Defines the text that will be displayed on the button. The default value is "Button".
- `disabled` (Boolean, optional)
