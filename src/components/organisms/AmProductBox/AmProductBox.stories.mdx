# AmProductBox

The `AmProductBox` component is used to display a product box.

## Props

The component accepts the following props:

- `headerInformation` (Object, required): The header information of the product box.
- `products` (Array, required): The products to be displayed in the product box.

## Usage

```vue
<template>
  <AmProductBox
    :header-information="headerInformation"
    :products="products"
  />
</template>

<script>
import { ref } from "@vue/composition-api";
import AmProductBox from "@/components/organisms/AmProductBox/AmProductBox.vue";

export default {
  components: { AmProductBox },
  setup() {
    const headerInformation = ref({
      title: "New Product Title",
      emphasisText: "20% off",
      description: "New Product Description",
    });
    const products = ref([
      {
        id: 1,
        name: "Product 1",
        price: 100,
        url: "https://picsum.photos/700",
        description: "Product 1 description",
      },
    ]);

    return {
      headerInformation,
      products,
    };
  },
};
```
